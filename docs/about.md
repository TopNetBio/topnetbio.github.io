# About

## TopNetBio

TopNetBio is an informal home for documentation, tutorials, and curated examples around **topological** and **network** analyses of **biological** systems — with a current focus on neural connectomes.

This site is **not** the official home of the Connectome Analysis library. The canonical source and API reference live at:

- Source: <https://github.com/openbraininstitute/connectome-analysis>
- Reference docs: <https://openbraininstitute.github.io/connectome-analysis/>

What you'll find here is complementary: an entry-level Python path, narrative walk-throughs, and notes that don't fit naturally into an API reference.
