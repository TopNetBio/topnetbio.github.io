# TopNetBio

**Topological and network approaches to biological connectomes.**

This site hosts a guided **Python tutorial** for newcomers and a working guide to the [**Connectome Analysis**](connectome-analysis/index.md) library developed at the Open Brain Institute.

## What is a connectome?

A **connectome** is a map of connections in a nervous system — neurons or brain regions as nodes, synapses or tracts as edges. Studying its structure with tools from graph theory and algebraic topology reveals organising principles that simple summary statistics miss.

The pages here are practical: load a matrix, run an analysis, read the result.
