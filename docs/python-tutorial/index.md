# Python Tutorial

A pragmatic, fast path from "I have Python installed" to "I just ran my first connectome analysis." Aimed at biologists, students, and researchers who want to use Python as a tool, not study it as a subject.

## Who this is for

- You can use a terminal, even if cautiously.
- You may have written Python before in scripts or notebooks, or you may be brand new.
- Your goal is to **load data, transform it, and produce a result** — not to become a software engineer.

## Who this is not for

- If you are looking for an in-depth Python language reference, see the [official Python tutorial](https://docs.python.org/3/tutorial/).
- If you already know Python *and* network science well, skip to the [Connectome Analysis Quickstart](../connectome-analysis/quickstart.md).

## How to use this tutorial

- **Type, don't copy.** Copy-pasting is faster but you'll learn less. Re-typing forces you to notice the parts you didn't understand.
- **Break things on purpose.** Comment out a line, change a value, see what happens. Error messages are teachers.
- **Keep a notebook of your own.** A scratchpad you can search later is more useful than this site.
